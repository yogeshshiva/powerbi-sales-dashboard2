# 📊 Sales Performance Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive Sales Performance Dashboard built using Power BI to analyze business data for February–March 2026.

The dashboard provides insights into revenue, profit, product performance, vendor contribution, and sales trends.

---

## 🔍 Key Insights

* Profit margin is approximately 64%
* Identification of top-performing and low-performing products
* Vendor-level analysis highlights major contributors
* Sales trend shows fluctuations over time

---

## 🛠 Tools & Technologies

* Power BI
* Power Query (Data Cleaning)
* DAX (Data Analysis Expressions)

---

## 📊 Dashboard Features

* KPI Cards (Revenue, Quantity, Cost, Profit, Profit %)
* Interactive filters (Date & Invoice Type)
* Region-wise sales analysis
* Product performance tracking
* Vendor performance analysis
* Time-series sales trend

---

## 📁 Dataset

* Internal sales dataset (cleaned and transformed using Power Query)

---

## 🎯 Objective

To transform raw sales data into actionable insights and support data-driven decision-making.
